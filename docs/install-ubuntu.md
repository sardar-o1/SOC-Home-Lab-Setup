# Install Ubuntu Server 

1. Go to the Ubuntu download page and download the latest LTS ISO for your architecture: https://ubuntu.com/download/desktop

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu1.png)

2. In VirtualBox, click the "New" button.

Give the VM a name, select the downloaded Ubuntu ISO as the installation image, and proceed with the unattended installation using the default settings to simplify the process.

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu2.png)

3. Next, create an Ubuntu username and password, and leave the OS installation options at their default values.

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu3.png)

4. Now specify the virtual hardware: assign 4 GB of RAM and 2 CPU cores (adjust based on your host's available memory and CPU).

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu4.png)

5. I gave the VM a 30 GB virtual hard disk, then clicked "Finish."

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu5.png)

6. Select the Ubuntu VM and click "Start". When the VM boots, choose "Try or Install Ubuntu" to begin the installation.

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu6.png)

7. The installation may take some time to complete.

![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu7.png)

8. Now log in and start using Ubuntu.
   
![img](https://github.com/sardar-o1/SOC-Home-Lab-Setup/blob/f552b4983da9a39c348afe1a53f43fe20ee1d9f1/Screenshots/ubuntu8.png)
